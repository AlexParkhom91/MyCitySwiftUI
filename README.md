# MyCitySwiftUI

Creating a app for tourism. 
The app includes weather forecast, city news portal webpage, points of interest and information about cafes and restaurants.

## Framework

```ruby
SwiftUI
MapKit
Foundation
WebKit
API
```

## Architecture pattern

```ruby
ModelViewViewModel(MVVM)
```
## Interface

```ruby
Programmatically
```

## Used in development
```ruby
MapKit
WebWiew
API
```
